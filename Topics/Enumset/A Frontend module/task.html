A Frontend module

Your team is creating a Java module that returns an ArrayList with an enum set (or sets) to apply the styles on the respective document type. The next part of the module has to be completed by you.

Consider the following info: Main-Heading means the main heading of the document and Quote|Bold signifies a quotation that is bold, similarly the switch cases represents various other segments of a document.


Sample Input:
Main-Heading

Sample Output:
[BOLD, LARGE, NORMAL]

Time limit: 5 seconds
